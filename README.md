# Control Dashboards

Applicazione Electron con stack web reattivo utilizzando React.

## Installazione

1. Clona il repository.
2. Esegui `npm install` per installare le dipendenze.

## Sviluppo

Per avviare l'app in modalità sviluppo:

```bash
npm start
```

Questo avvierà Vite per il frontend e Electron per l'app desktop.

## Build

Per costruire l'app per produzione:

```bash
npm run build
npm run electron
```

## Piattaforme

Questa app è progettata per funzionare su Windows. Assicurati di avere Node.js e npm installati.

## Tecnologie

- Electron
- React
- Vite
